  • cd: Change Directory
  • ls: List Files
  • rm -rf [dirName]: Recurively removes directories and all sub-directories with “-r”. “-f” does not prompt for each child directory.
  • lsusb: Lists USB devices
  • fdisk -l: Lists filesystems and their mount points on root
  • mount: Used to attach filesystems to root
  • umount: Detaches a filesystem from root
  • tar -zxvf: Untar to current directory (-t list file contents -x extract -f use archive file -v verbosely lists files processed -z gzip)
  • df -h: Show disk space available in 'human readable' format
  • run startup script (debian) - Edit /etc/rc.local
  • uname -r: Shows OS Kernel
  • kill -9 pid: Kills a process by its PID
  • adduser : runs the add user script
  • /etc/sudoers: Configuration file which grants a standard user the ability to run sudo commands

    • Network

    • ifconfig: Display information on Local Network Adapters
    • ifup/down : Brings an interface up or down
    • ifconfig netmask : Assigns a static IP address to a selected interface
    • route add default gw - Add a default gateway
    • dhclient : Sends/Renews a dhcp request
    • arp -s : Enter a static ARP entry
    • /etc/resolv.conf: Configuration file for nameserver entries. Example entry - nameserver 161.76.9.1

    Assign a static IP at startup example:

    • /etc/networking/interfaces: Configuration file for network interfaces. Lines beginning with auto “are used to identify the physical interfaces to be brought up when ifup is run with the -a option. (This option is used by the system boot scripts.)”. Auto just brings an interface up at startup
    • auto eth0 #Initialises eth0 on startup
    • iface eth0 inet static #Static Configuration
    • address 192.168.0.42
    • network 192.168.0.0
    • netmask 255.255.255.0
    • broadcast 192.168.0.255
    • gateway 192.168.0.1

      • for DHCP

    • auto eth0
    • iface eth0 inet dhcp
